Difference between width:auto and width:100%

  • Example without any width property(auto/100%) on Child Container

    Here child container floats outside the parent container.

    CSS Box properties of Parent Container:

    • width: 320px;
    • padding: 20px;
    • border: 5px dotted #000;
    • margin: 10px;

    Total Width = Left-Margin + Left-Border + Left-Padding + Content-Width + Right-Padding + Right-Border + Right-Margin
           = 10 + 5 + 20 + 320 + 20 + 5 + 10
           = 390px;

    CSS Box properties of Child Container:

    • width: 320px;
    • padding: 20px;
    • border: 5px dotted #000;
    • margin: 10px;

    Total Width = Left-Margin + Left-Border + Left-Padding + Content-Width + Right-Padding + Right-Border + Right-Margin
           = 10 + 5 + 20 + 320 + 20 + 5 + 10
           = 390px;

  • Example with the property width:auto on Child Container

    Here the width of the child container does not consider padding, border and margin pixels. It calculates the content of the width in such a way that the child container is inside the parent container and does not float outside.

    To find out how it calculating with the width:auto, refer the article - https://www.w3.org/TR/CSS21/visudet.html#blockwidth

    CSS Box properties of Parent Container:

    • width: 320px;
    • padding: 20px;
    • border: 5px dotted #000;
    • margin: 10px;

    Total Width = Left-Margin + Left-Border + Left-Padding + Content-Width + Right-Padding + Right-Border + Right-Margin
           = 10 + 5 + 20 + 320 + 20 + 5 + 10
           = 390px;

    CSS Box properties of Child Container:

    • width: auto;
    • padding: 20px;
    • border: 5px dotted #000;
    • margin: 10px;

    Total Width = Left-Margin + Left-Border + Left-Padding + Content-Width + Right-Padding + Right-Border + Right-Margin
           = 20 + 5 + 20 + 230 + 20 + 5 + 20
           = 320px;

  • Example with the property width:100% on Child Container

    Here the width of the child container considers padding, border and margin pixels. It calculates the content of the width by adding padding, border and margin. Finally the child container floats outside the parent container.

    To find out how it calculating with the width:100%, refer the article - xxxx
